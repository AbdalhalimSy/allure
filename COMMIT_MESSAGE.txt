- Split monolithic `account.json` (21KB) into 5 focused modules
- Removed `accountSettings` wrapper namespace
- Updated all components to use new clean namespaces

## New Structure
```
account-main.json (1.7KB)       → account.*
account-basic.json (1.9KB)      → accountBasic.*
account-appearance.json (1.2KB) → accountAppearance.*
account-billing.json (2.8KB)    → accountBilling.*
account-security.json (1.1KB)   → accountSecurity.*
```

## Migration
Old pattern: `t('accountSettings.account.billing.status.active')`
New pattern: `t('accountBilling.status.active')`
